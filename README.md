# Tier-List-Acad-mico
Tier List com colunas de Ensino, Pesquisa e Extensão, para gameficação do ensino da tríade da universidade.

O objetivo é realizar explicações do que pode ser feito em cada uma destas bases da universidade. Este tier list pode ser empregado em complemento a uma atividade de world café, trazendo como post-it as sugestões escritas pelos estudantes nos cartazes.

Tier list gerado com auxílio do Large Language Model (LLM) Anthropic Claude.
